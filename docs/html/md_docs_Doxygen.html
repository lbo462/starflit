Starflit
Arduino project that aims to organize a fleet of strandbeest-like robots.
Generating the documentation

Here, we use the famously know Doxygen to generate out documentation.

To use it properly, install it for your OS by following the instruction on their website.

In order to generate the graphs as well, one should have the dot command available. For that purpose, install Graphviz and make sure it's reachable before you call Doxygen.

Then, use the Doxyfile file to generate the documentation.

On Linux, call doxygen Doxyfile at the root of this project.

It will create / update the folder docs/html/ that will contain the generated doc. Open docs/html/index.html in your browser to read through it.
