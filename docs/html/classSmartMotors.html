Detailed Description

Smart motors is abstract representation of motors monitored via control engineer.

Instances of this class have a motor and an axelgyro module. The motor module makes the motors move by applying voltage ; The axelgyro module look out for any drifting behaviour and adapt the motor commands.

◆ pidSetpoint()

void SmartMotors::pidSetpoint ( double  setpoint)
private

Resets the SetPoint for the PID.

This can't be done easily through pid.SetPoint(). Instead, some weird code happens inside this function to make it work properly. It resets the internal sums of the PID object, then sets the setPoint.
